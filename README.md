# HireMind AI

HireMind AI is a smart job tracker and career assistant built with the MERN stack. It helps users explore jobs, search and filter listings, save favourite roles, manage saved jobs, switch between dark and light mode, and get AI-inspired career insights.

## Overview

This project is designed to demonstrate modern full-stack web development using React on the frontend It focuses on practical features such as routing, state management, reusable UI components, and beginner-friendly AI integration for career guidance.

## Features

- Explore job opportunities in a clean and modern UI.
- Search jobs by title, company, or skills.
- Filter jobs by category and work mode.
- View detailed job descriptions, responsibilities, and requirements.
- Save and unsave favourite jobs.
- Manage saved jobs from a dedicated page.
- Switch between dark mode and light mode.
- View AI-style career insights and recommendations.
- Modular project structure for scalability and learning.

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/RCThakur/HireMind-AI.git
cd HireMind-AI
```

### 2. Install frontend dependencies

```bash
cd client
npm install
```

## Run the Project

### Start the frontend

```bash
cd client
npm run dev
```
